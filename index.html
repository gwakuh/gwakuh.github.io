TAP Device

2021-04-29
Network Programming

TAP(Terminal Access Point) Device Kernel Virtual Network Device 물리적인 네트워크 인터페이스가 아닌 가상의 인터페이스를 생성할 때 사용한다. 인터페이스 생성 OS 버전: CentOS 7 iproute version 4.11.0 # ip tuntap add mode tap tap1 # ip link set tap1 up iproute 2.6.34 이전 버전의 경우 ip tuntap 명령을 지원하지 않음 패킷 전송 처음엔 raw socket을 만들어 sendto 함수로 전송하는 로직으로 시도하였으나, TAP Device를 UP 시켜도 LOWER_UP 상태가 아니라 전송이 안되는 것으로 보인다.
